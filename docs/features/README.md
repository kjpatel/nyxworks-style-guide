# Features Backlog

This directory tracks feature ideas from planning through release.

## Structure

- `planned/`: Not started.
- `in-progress/`: Actively being implemented.
- `shipped/`: Completed and released.

Move files between folders as status changes.

## File Naming

Use: `YYYY-MM-DD-short-slug.md`

Example: `2026-03-11-dark-mode-toggle.md`

## Authoring

1. Copy `_template.md`.
2. Fill in all sections.
3. Add the file under `planned/`.
4. Move it to `in-progress/` and `shipped/` over time.
